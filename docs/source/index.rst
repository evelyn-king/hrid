hrid documentation
==================

Generate human readable ids using python.

Purpose
-------

Heavily inspired by `Asana's naming scheme <https://asana.com/inside-asana/6-sad-squid-snuggle-softly>`_,
this package contains the tools to generate random phrases that represent nearly-unique
human readable ids for objects.

Installation
------------

This package is currently in development and must be installed using
a dev install

.. code-block:: console
   :caption: Installation

   git clone git@github.com/evelyn-king/hrid.git
   cd hrid
   python -m pip install -e ".[all]"

After that, you should be ready. Happy hacking!
